# Work Journal

Локальный рабочий журнал с поддержкой фотографий.

## Быстрый старт

### Вариант 1: Только index.html (без сервера)
Откройте **index.html** двойным щелчком. Работает всё, кроме загрузки фото на сервер: при прикреплении фото сохраняется **только ссылка** (имя файла). Файл нужно положить в папку **photos/** рядом с index.html — тогда фото будет отображаться. Интернет не нужен.

### Вариант 2: С локальным сервером
**Требования:** Node.js 16+ (https://nodejs.org)

```bash
node server.js
```
Откройте браузер: **http://localhost:3000**

Фото можно загружать через интерфейс — они сохраняются в папку `photos/`. Никаких npm install — сервер использует только встроенные модули Node.js.

## Структура проекта

```
work-journal/
├── server.js          ← локальный сервер (запустить один раз)
├── index.html         ← интерфейс приложения
├── photos/            ← фото хранятся здесь (создаётся автоматически)
├── css/
│   └── styles.css
├── js/
│   ├── db.js          ← IndexedDB: записи, категории, снимки
│   ├── i18n.js        ← менеджер языков
│   ├── photos.js      ← загрузка фото, галерея, лайтбокс
│   ├── report.js      ← генерация отчётов и печать
│   ├── snapshot.js    ← автоматические снимки базы
│   └── app.js         ← основная логика
└── i18n/
    ├── ru.js           ← Русский
    ├── en.js           ← English
    └── pl.js           ← Polski
```

## Перенос на другой компьютер

1. Скопируйте всю папку `work-journal/` вместе с `photos/`
2. На новом компьютере запустите `node server.js`
3. Или: используйте **Экспорт** в приложении (фото включаются в JSON как base64)

## Фотографии

- Форматы: JPEG, PNG, GIF, WebP
- Максимум 10 фото на запись
- **Без сервера** (только index.html): при добавлении фото сохраняется ссылка на файл; сам файл нужно положить в папку **photos/** проекта — тогда картинка будет отображаться.
- **С сервером**: фото загружаются через интерфейс и сохраняются в папку `photos/` (до 10 MB на файл).
- Во всех случаях в записях хранится только ссылка (имя файла); файлы лежат в папке `photos/`.

## База данных

- IndexedDB в браузере (записи, категории, снимки архива)
- Автоматический снимок перед каждой очисткой и импортом
- Экспорт/импорт через JSON

## Язык интерфейса

Переключается в **Настройки** → Язык: 🇷🇺 Русский / 🇬🇧 English / 🇵🇱 Polski.
Данные (записи) языком не затрагиваются.
